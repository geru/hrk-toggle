Installing Y

The PHP YAML extension is very easy-to-use. It is written in C, so it is also very fast.

Here is a guide for installing the PHP YAML extension.

If you will be running Y on a server that lacks the YAML extension, you can still do so provided you first invoke your webserver from a development machine that has the YAML extension installed. When Y locates YAML files, it translates them to JSON, just in case it might be usefule. If you add the JSON files to your website source code and upload it to your server, the server will find and use the JSON files. This is the technique needed to use Y with Pantheon web-hosting.
